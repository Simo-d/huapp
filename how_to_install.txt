# Guide de démarrage rapide – HU Management System

## Prérequis
- Node.js installé (version 14 ou supérieure)
- Deux fenêtres de terminal/commande ouvertes (une pour le backend, une pour le frontend)

---

## 🔧 Installation initiale (à faire une seule fois)

### Terminal 1 – Backend
cd E:\GeniePot\aiprojects\huapp\server
npm install

### Terminal 2 – Frontend
cd E:\GeniePot\aiprojects\huapp
npm install

---

## ▶️ Lancement de l’application

### Étape 1 – Démarrer le serveur backend (Terminal 1)
cd E:\GeniePot\aiprojects\huapp\server
node index.js

Sortie attendue :
🚀 Server is running on port 5000
📡 API available at http://localhost:5000/api

### Étape 2 – Démarrer le frontend React (Terminal 2)
cd E:\GeniePot\aiprojects\huapp
npm start

👉 L’application s’ouvre automatiquement sur http://localhost:3000

---

## ⚠️ Dépannage
- Serveur backend ne démarre pas :
  1. Vérifiez que le port 5000 est libre
  2. Supprimez node_modules puis exécutez npm install

- Frontend React ne démarre pas :
  1. Vérifiez que le port 3000 est libre
  2. Supprimez node_modules puis exécutez npm install

- Erreur de base de données :
  - Démarrez toujours le backend AVANT le frontend
